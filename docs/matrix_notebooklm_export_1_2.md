# NotebookLM Export 1.2 Spec-to-Implementation Matrix

## Objective

Deliver NotebookLM Export 1.2: scan starts only after the user clicks Scan, NotebookLM structural rich-text wrappers no longer flatten headings, tables, and adjacent blocks in Markdown exports, saved-page formulas export as Markdown LaTeX when supported source or KaTeX visual DOM is available, and adjacent inline Markdown tokens remain parseable by Typora-style renderers.
